### （1）使用script 标签引入:
``` javascript
<script type="text/javascript" src="ztpl.js"></script>
<script type="text/javascript">
	ztpl('#container', data);
</script>
```
### （2）使用requireJs异步引入:
``` javascript
require(['ztpl'], function (ztpl) {
	ztpl('#container', data);
});
```
### （3）使用ES6方式导入：
``` javascript
import ztpl from './ztpl.js';
ztpl('#container', data);
```
